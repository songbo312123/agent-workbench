# Agent Workbench 前端实现规范

## 1. 前端目标

前端负责把多智能体执行过程动态、清晰、可追踪地展示出来。

核心要求：

```text
1. 实时订阅事件流
2. 由事件 reducer 推导状态
3. 显示 Agent 状态变化
4. 显示并行泳道
5. 显示 Trace Console
6. 支持点击事件查看详情
7. 未来支持任务图和回放
```

## 7. Event Reducer

所有状态变化都通过 reducer。

```ts
export function reduceEvent(
  state: WorkbenchState,
  event: TraceEvent
): Partial<WorkbenchState> {
  const agents = { ...state.agents };
  const tasks = { ...state.tasks };
  const events = appendUniqueEvent(state.events, event);
  let session = state.session;

  switch (event.type) {
    case "session_started":
      session = {
        id: event.sessionId,
        userTask: String(event.payload.userTask || ""),
        status: "planning",
        startedAt: event.timestamp
      };
      break;

    case "planning_started":
      if (event.agentId && agents[event.agentId]) {
        agents[event.agentId] = {
          ...agents[event.agentId],
          status: "planning",
          currentAction: event.summary
        };
      }
      break;

    case "plan_created":
      for (const rawTask of event.payload.tasks as AgentTask[]) {
        tasks[rawTask.id] = {
          ...rawTask,
          status: rawTask.dependsOn.length ? "waiting" : "pending"
        };
      }
      if (session) session = { ...session, status: "running" };
      break;

    case "task_started":
      if (event.taskId && tasks[event.taskId]) {
        tasks[event.taskId] = {
          ...tasks[event.taskId],
          status: "running",
          startedAt: event.timestamp
        };
      }
      if (event.agentId && agents[event.agentId]) {
        agents[event.agentId] = {
          ...agents[event.agentId],
          status: "running",
          currentTaskId: event.taskId,
          currentAction: event.summary
        };
      }
      break;

    case "tool_call_started":
      if (event.agentId && agents[event.agentId]) {
        agents[event.agentId] = {
          ...agents[event.agentId],
          status: "tool_call",
          currentAction: event.summary
        };
      }
      break;

    case "tool_call_finished":
      if (event.agentId && agents[event.agentId]) {
        agents[event.agentId] = {
          ...agents[event.agentId],
          status: "running",
          currentAction: event.summary
        };
      }
      break;

    case "agent_message_delta":
      if (event.agentId && agents[event.agentId]) {
        agents[event.agentId] = {
          ...agents[event.agentId],
          latestMessage: String(event.payload.text || "")
        };
      }
      break;

    case "task_finished":
      if (event.taskId && tasks[event.taskId]) {
        tasks[event.taskId] = {
          ...tasks[event.taskId],
          status: "success",
          finishedAt: event.timestamp,
          result: String(event.payload.result || "")
        };
      }
      if (event.agentId && agents[event.agentId]) {
        agents[event.agentId] = {
          ...agents[event.agentId],
          status: "success",
          currentAction: "任务完成"
        };
      }
      break;

    case "task_failed":
      if (event.taskId && tasks[event.taskId]) {
        tasks[event.taskId] = {
          ...tasks[event.taskId],
          status: "failed",
          finishedAt: event.timestamp,
          error: String(event.payload.errorMessage || event.payload.error || "")
        };
      }
      if (event.agentId && agents[event.agentId]) {
        agents[event.agentId] = {
          ...agents[event.agentId],
          status: "failed",
          currentAction: event.summary
        };
      }
      break;

    case "session_finished":
      if (session) {
        session = {
          ...session,
          status: "completed",
          finishedAt: event.timestamp,
          finalAnswer: String(event.payload.finalAnswer || "")
        };
      }
      break;
  }

  return { session, agents, tasks, events };
}
```

## 8. 实时事件客户端

MVP 可选 WebSocket 或 SSE。

推荐先用 SSE：

```ts
export function subscribeSessionEvents(
  sessionId: string,
  onEvent: (event: TraceEvent) => void
) {
  const source = new EventSource(`/api/sessions/${sessionId}/events/stream`);

  source.addEventListener("agent_event", message => {
    const event = JSON.parse(message.data) as TraceEvent;
    onEvent(event);
  });

  source.onerror = () => {
    source.close();
  };

  return () => source.close();
}
```

如果需要双向控制更多，使用 WebSocket。

```ts
function groupEventsByAgent(events: TraceEvent[]) {
  return events.reduce<Record<string, TraceEvent[]>>((acc, event) => {
    if (!event.agentId) return acc;
    acc[event.agentId] ??= [];
    acc[event.agentId].push(event);
    return acc;
  }, {});
}
```

### 10.3 x 坐标计算

MVP：

```text
x = (event.sequence - firstSequence) * 96px
```

### 10.4 事件块标签

```ts
function getEventLabel(event: TraceEvent): string {
  if (event.type === "tool_call_started") {
    return `工具：${String(event.payload.toolName || "")}`;
  }
  if (event.type === "agent_message_delta") return "生成内容";
  if (event.type === "task_started") return "开始任务";
  if (event.type === "task_finished") return "完成";
  if (event.type === "task_failed") return "失败";
  return event.summary || event.type;
}
```

## 14. Mock Event Player

第一阶段必须实现 mock player。

```ts
export function playMockEvents(
  events: TraceEvent[],
  onEvent: (event: TraceEvent) => void
) {
  const timers: number[] = [];

  const first = events[0]?.sequence ?? 0;

  for (const event of events) {
    const delay = Math.max(0, event.sequence - first) * 600;
    const timer = window.setTimeout(() => onEvent(event), delay);
    timers.push(timer);
  }

  return () => timers.forEach(window.clearTimeout);
}
```

即使后面接真实后端，也要保留 mock player，便于 UI 开发和测试。

## 15. 前端验收标准

```text
1. 点击开始后 2 秒内出现动态
2. Agent 卡片状态实时变化
3. 至少两个 Agent 在 LaneView 中并行显示
4. 工具调用事件有明显视觉状态
5. 点击 Agent 显示详情
6. 点击 Event 显示 payload
7. TraceConsole 自动追加日志
8. session_finished 后展示最终结果
9. 不接后端时 mock event 也能完整跑通
10. 接后端时只替换事件来源，不改 UI 架构
```
